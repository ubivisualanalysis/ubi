# 查看趋势

表格中，各维度值下的指标，可查看过往数据趋势  


#### 操作方法

1. 鼠标移入单元格，出现查看趋势&对比控件
2. 点击查看趋势，进入新标签页

{% hint style="info" %}

{% endhint %}
