# 添加对比

表格中，各维度下的指标趋势，也可多个同时查看，进行对比

#### 操作方法

1. 进行维度设置后，鼠标移入单元格，单元格下方出现查看趋势&对比控件
2. 点击“添加对比”，所添加指标进入页面右上角的“对比框”中
3. 添加多个需对比的指标后，点击对比框中右下角“对比”，进入新标签页

{% hint style="info" %}

{% endhint %}

如添加失误，可点击单独删除指标，也可点击“清空“对比框

当对比框遮挡报表关键信息时，可鼠标移入对比框进行拖动
